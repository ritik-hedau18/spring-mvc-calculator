# spring-mvc-calculator

## Description
A simple calculator web application built using Spring MVC
to understand request mapping, form handling, and MVC flow.

## Technologies Used
- Java
- Spring MVC
- JSP
- Maven
- Apache Tomcat

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Learning Outcome
- Understood MVC architecture
- Form submission handling
- Controller to JSP data flow
